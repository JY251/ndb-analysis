● ITS（Interrupted Time Series）解析のレベル変化と傾き変化

  ITS解析では、介入前後の変化を2種類で評価します。

  ---
  1. レベル変化（Level change / 切片変化）

  介入直後に即座に起きるアウトカムの「段差」です。

  アウトカム
      |         ← 介入点
      |    _____|‾‾‾‾‾‾  ← 介入後のライン
      |___/               ← 介入前のライン
      |
      +—————————————————→ 時間

  - 介入前トレンドの延長線と、介入後の実測値の垂直方向の差
  - 回帰モデルでは D_t（介入ダミー変数, 0→1）の係数に対応

  ---
  2. 傾き変化（Slope change / トレンド変化）

  介入後に徐々にトレンド（傾き）が変わることです。

  アウトカム
      |              /  ← 介入後：傾きが増加
      |         /‾‾‾
      |    /‾‾‾  ← 介入点
      |___/
      +—————————————————→ 時間

  - 介入前後で単位時間あたりの変化量（傾き）の差
  - 回帰モデルでは T_t × D_t（時間 × 介入ダミー）の係数に対応

  ---
  典型的なITS回帰モデル

  Y_t = β₀ + β₁·T + β₂·D + β₃·(T×D) + ε

  ┌──────┬──────────────────────────────────┐
  │ 係数 │               意味               │
  ├──────┼──────────────────────────────────┤
  │ β₀   │ 介入前ベースライン（切片）       │
  ├──────┼──────────────────────────────────┤
  │ β₁   │ 介入前の傾き（トレンド）         │
  ├──────┼──────────────────────────────────┤
  │ β₂   │ レベル変化（介入直後の段差）     │
  ├──────┼──────────────────────────────────┤
  │ β₃   │ 傾き変化（介入前後のトレンド差） │
  └──────┴──────────────────────────────────┘

● ITS解析：Single group vs Control group

  ---
  Single Group ITS（単一群）

  介入を受けた1グループのみで前後を比較します。

  アウトカム
      |         ↑介入
      |    ____/‾‾‾‾‾‾‾‾  ← 介入群（1本のみ）
      |___/
      +—————————————————→ 時間

  - シンプルだが、外部要因（季節性・政策変化など）の影響を排除できない
  - 「介入がなくても変化していたかも」という反事実が不明

  ---
  Control Group ITS（対照群あり）

  介入を受けていないグループを同時に観察し、差分を取ります。

  アウトカム
      |         ↑介入
      |    ____/‾‾‾‾‾‾‾‾  ← 介入群
      |___/‾‾‾‾‾‾‾‾‾‾‾‾‾  ← 対照群（介入なし）
      +—————————————————→ 時間

  - 外部要因は両群に同様に影響すると仮定 → 差分で除去できる
  - より因果推論に近い推定が可能（差分の差分：DiD的な考え方）

  ---
  モデルの違い

  Single group
  Y_t = β₀ + β₁·T + β₂·D + β₃·(T×D) + ε

  Control group（差分モデル）
  Y_t = β₀ + β₁·T + β₂·D + β₃·(T×D)
          + β₄·Group + β₅·(Group×T)
          + β₆·(Group×D) + β₇·(Group×T×D) + ε

  ┌──────┬────────────────────────────────┐
  │ 係数 │              意味              │
  ├──────┼────────────────────────────────┤
  │ β₆   │ 群間のレベル変化の差           │
  ├──────┼────────────────────────────────┤
  │ β₇   │ 群間の傾き変化の差（主な関心） │
  └──────┴────────────────────────────────┘

  ---
  選び方

  ┌────────────────────────────────────────────┬───────────────┐
  │                    状況                    │     推奨      │
  ├────────────────────────────────────────────┼───────────────┤
  │ 外部要因の影響が小さい・対照群が存在しない │ Single group  │
  ├────────────────────────────────────────────┼───────────────┤
  │ 外部要因が懸念される・比較可能な群がある   │ Control group │
  ├────────────────────────────────────────────┼───────────────┤
  │ より強い因果推論が必要                     │ Control group │
  └────────────────────────────────────────────┴───────────────┘

  対照群に何を使うか（地域・年齢層・未介入施設など）も重要です。具体的な研究設計
  はありますか？
