# NDB注射月別データ バイオシミラー抽出スクリプト

## 概要

`make_bs_files.py` は、NDBオープンデータ（注射薬・診療月別）から
バイオシミラー（BS）に該当する行を抽出・ハイライトするスクリプトです。

---

## 対象ファイル

| ラベル | ファイル | 期間 |
|--------|----------|------|
| rd10 | `rd10_injection_monthly.xlsx` | FY2023（2023年4月〜2024年3月） |
| rd11 | `【注射】診療月別薬効分類別数量(公費含む).xlsx` | FY2024（2024年4月〜2025年3月） |

各ファイルには以下の3シートが含まれます：
- 注射薬 外来（院内）
- 注射薬 外来（院外）
- 注射薬 入院

---

## 出力ファイル

| ファイル名 | 内容 |
|------------|------|
| `rd10_injection_monthly_highlighted_v2.xlsx` | FY2023：BS行を黄色ハイライト（元データ全行維持） |
| `rd10_injection_monthly_BS_only_v2.xlsx` | FY2023：BS行のみ抽出（列A・B前方補完済み） |
| `rd11_injection_monthly_highlighted_v2.xlsx` | FY2024：BS行を黄色ハイライト（元データ全行維持） |
| `rd11_injection_monthly_BS_only_v2.xlsx` | FY2024：BS行のみ抽出（列A・B前方補完済み） |

---

## バイオシミラー判定ロジック

医薬品名（列D）に全角「ＢＳ」を含む行をバイオシミラーとして判定します。

```python
"ＢＳ" in 医薬品名
```

ただし、以下に該当する行は **除外** します（後述の注意事項参照）。

---

## ⚠️ 重要な注意事項：薬効分類341の除外

### 問題の内容

薬効分類 **341（人工腎臓透析用剤）** に含まれる以下の製品は、
医薬品名に「ＢＳ」を含むが、**バイオシミラーではない**：

| 医薬品名 | 分類 |
|----------|------|
| サブラッド血液ろ過用補充液ＢＳＧ　2020mL | 341 |
| サブラッド血液ろ過用補充液ＢＳＧ　1010mL | 341 |

### 「ＢＳＧ」とは

「ＢＳＧ」は **Balanced Salt with Glycol** に由来する製品略称であり、
透析用補充液の組成を示すものです。後続生物学的製剤（バイオシミラー）の
命名規則である「（一般名）ＢＳ（メーカー名）」とは全く異なります。

### 対応

スクリプト内の `EXCLUDE_CATEGORIES` に `341` を明示的に追加して除外しています：

```python
EXCLUDE_CATEGORIES = {341}  # 人工腎臓透析用剤（サブラッドＢＳＧ）
```

将来的にデータが更新された際も、同様の確認を行うことを推奨します。

## 列構造の補足

元のExcelデータでは列A（薬効分類コード）・列B（薬効分類名称）は
各カテゴリグループの**先頭行にのみ**値が入っており、
同グループの後続行はブランクになっています。

BS抽出版では、これを**前方補完**して全行に分類情報を付与しています。
